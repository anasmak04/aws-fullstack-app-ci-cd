# CI/CD AWS Deployment Project

---

## 1️⃣ Objectif du projet

Déployer une application web full-stack sur AWS avec :

- **WebApp Frontend** (Angular) sur EC2 public
- **Backend API** (Node.js/Express) sur EC2 public
- **Base de données MySQL** sur EC2 privé
- **Monitoring** avec CloudWatch
- **Notifications** SNS pour les alertes
- **Pipeline CI/CD** via GitHub Actions et CloudFormation

### Compétences évaluées

- ✅ AWS CLI & CloudFormation
- ✅ EC2, Security Groups, VPC, Subnets
- ✅ S3, IAM roles
- ✅ CI/CD pipeline avec GitHub Actions
- ✅ Monitoring et alertes
